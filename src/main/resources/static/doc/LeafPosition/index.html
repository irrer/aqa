This test determines the position error of each collimator leaf at a variety of positions. For each leaf position, the expected (from RTPLAN) position is subtracted from the observed position and the error reported in mm in the isoplane:

error = observed - expected

The following example shows the measured positions of the leaf ends and sides (as dashed lines) as well as the expected (planned) positions, compensated with the results from the collimator centering, shown as solid lines.
